# 第一次作业
## 作业内容
给定一段音频，请提取12维MFCC特征和23维FBank，阅读代码预加重、分帧、加窗、快速傅里叶变换部分，完善作业代码中FBank特征提取和MFCC特征提取部分，并给出最终的FBank特征和MFCC特征，存储在纯文本中，并观察画出的频谱图，文本行数为语音帧数，每一行表示一帧语音的特征，各维度之间空格分隔
## 代码文件说明
代码依赖

    python3
    librosa
